## Overview

This is the **back-end API service** for the Stock Management System assignment. It provides RESTful endpoints to support product and order management functionalities used by the React front-end application.

---

## Responsibilities

* Serve product data for listing, filtering, and detail views
* Handle product updates
* Serve order data with filtering and status-based sorting

---

## Setup & Run

1. Download and open the project in **Visual Studio Code**.
2. Install dependencies:

```bash
npm install
```

3. Start the server:

```bash
npm run dev
```

---

## Usage

* Ensure this back-end service is running **before** starting the front-end application.
* The front-end communicates with this API to fetch and update products and orders.

---

## Technologies Used

* Node.js
* Express.js
* REST API

---

## Notes

This back-end is intentionally kept simple to support the assignment requirements and demonstrate API integration with a React front-end.
